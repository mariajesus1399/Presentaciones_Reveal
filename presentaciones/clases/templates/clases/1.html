Ejercicio

Se generan 1000 números, de forma aleatoria (distribución uniforme) entre 0 y 1. Calcule la probabilidad de que un numero dado sea mayor a 0,7.

Solución

Debido a que se tiene una distribución uniforme, se puede asumir que de los 1000 números, unos 300 son mayores a 0,7. Podemos entonces calcular la probabilidad de este evento: \[\begin{aligned} P(A) & = \frac{300}{1000} = 30\% \\ \end{aligned} \]

Simulación

  1. Generar una distribución uniforme de 1000 datos con valores entre 0 y 1
  2. Obtener la cantidad de datos mayores a 0,7.
  3. Calcular la probabilidad

Función de solución


			def get_probs (elements, x):
				greater = 0
				lesser = 0
				# Recorremos los elementos y los clasificamos
				for element in elements:
					if element > x:
						greater += 1
					else:
						lesser += 1
				# Calculamos las probabilidades correspondientes
				return greater/len(elements), lesser/len(elements)
